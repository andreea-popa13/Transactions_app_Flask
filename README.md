# Transactions_app_Flask

A simple web application built with Flask for managing financial transactions.

## Features

- View all transactions
- Add new transactions
- Edit existing transactions
- Delete transactions
- Search transactions by amount range
- Display the total balance

## Technologies

- Python
- Flask
- HTML
- Bootstrap

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/flask-transaction-manager.git
   ```

2. Navigate to the project folder:
   ```bash
   cd flask-transaction-manager
   ```

3. Install Flask:
   ```bash
   pip install flask
   ```

4. Run the application:
   ```bash
   python app.py
   ```

5. Open your browser and go to:
   ```
   http://127.0.0.1:5000
   ```
